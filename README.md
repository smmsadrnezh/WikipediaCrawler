This is a simple web crawler for persian wikipedia based on scrapy and beautifulsoup.

How to run
=========

This plugin is based on [Scrapy](#). So install these packages with pip.

  * scrapy
  * beautifulsoup
  * requests
  * scipy
  * numpy
  * elasticsearch

How to use
=========

`scrapy crawl wikipedia 2>logs`
